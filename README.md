# AI Test API

A RESTful API server built with Express.js, PostgreSQL, featuring Swagger documentation and Docker support.

## Features

- 🚀 Express.js REST API
- 🐘 PostgreSQL database
- 🔐 JWT-based authentication
- 🎮 Blizzard OAuth integration
- 📚 Swagger/OpenAPI documentation
- 🐳 Docker & Docker Compose support
- 🔒 CORS enabled
- 📝 Request logging with Morgan
- ♻️ Environment variable configuration
- 🏥 Health check endpoint

## Prerequisites

- Node.js 18+ (or Docker)
- npm or yarn
